CSS

CSSson las siglas deCascading Style Sheets. que puede traducirse como hojas de estilo en cascada

El lenguaje HTMLse usa para diseñar las páginas web. Es muy sencillo modificar el código CSS de un sitio web, aunque la manera exacta varía según el programa utilizado y también puede cambiar entre distintas ediciones de un mismo software, pero suelen mantener un funcionamiento similar.

CSS se basa en reglas que se escriben dentro de llaves y siguen una sintaxis específica.

Historia de CSS

Los orígenes del lenguaje CSS se sitúan en el año 1996, cuando Håkon Wium Lie propuso su primera versión.
